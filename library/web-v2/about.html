📚 The Library

The Library is a private audiobook management system for organizing, converting, and streaming a personal audiobook collection. It provides a browser-based interface for browsing, searching, and listening to audiobooks from any device on the network.

Built with an Art Deco aesthetic inspired by the grand reading rooms of the 1920s, The Library combines modern web audio technology with classic design sensibilities — because a good book deserves a beautiful shelf.

★ Concept & Creation

The Library was conceived and designed by Bosco, who envisioned a self-hosted audiobook platform that treats a personal collection with the care and presentation it deserves — encrypted authentication, position syncing across devices, and an interface that makes browsing a pleasure rather than a chore.

Development is a collaborative effort between Bosco and Claude (by Anthropic). From architecture decisions and security hardening to the Art Deco visual language and audio playback engine, every feature has been shaped through genuine partnership — human vision guiding AI capability, each strengthening the other.

🌐 Localization & AI Services

Chinese (and other future language) support in The Library would not be possible without the generosity and craft of the open AI community and the GPU hosts who keep these models affordable to run. The Library extends heartfelt gratitude to:

DeepL

Neural machine translation for book descriptions, UI strings, and on-demand text translation — consistently producing the most natural Chinese renderings of English prose we tested.

OpenAI Whisper

Open-source speech-to-text model used to transcribe audiobook audio into source subtitles before translation. Accurate across accents, narration styles, and long-form listening.

Coqui XTTS

Multilingual voice cloning and text-to-speech synthesis, enabling translated narration that preserves the character of the original reader across languages.

Hugging Face

Home to the open model ecosystem that makes projects like this possible. The Library pulls Whisper, XTTS, and supporting models from the Hugging Face Hub — a public good maintained by a community that believes in open AI.

Vast.ai

Peer-to-peer GPU marketplace that runs Whisper transcription and XTTS narration on demand. Affordable cold-start GPU time is what makes per-book translation economically viable for a private library.

RunPod

Serverless GPU platform used as a secondary provider for longer narration jobs. Fast spin-up times and fair pricing keep The Library’s translated audio pipeline responsive.

🛠 Third-Party Attributions

The Library is built on the shoulders of excellent open-source tools and libraries:

FFmpeg

Audio conversion and metadata extraction. The backbone of the audiobook processing pipeline, handling format transcoding, chapter detection, and cover art extraction.

SQLCipher

Encrypted SQLite database providing transparent 256-bit AES encryption for user authentication data, session tokens, and account information.

Flask

Python web framework powering the API server and back-office administration interface. Lightweight, flexible, and battle-tested.

mutagen

Audio metadata parsing library for reading and writing tags across multiple audio formats including Opus, M4B, MP3, and FLAC.

PyOTP

TOTP-based authentication providing time-based one-time passwords compatible with standard authenticator apps.

FIDO2 / WebAuthn

Passkey authentication support enabling passwordless sign-in with hardware security keys, biometrics, and platform authenticators.

Howler.js

Audio playback engine providing reliable cross-browser audio with Web Audio API fallback, volume control, and seeking support.
